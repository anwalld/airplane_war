（1. 数据的完整设计



全局变量类
class AllGame{
int bgm;---放哪首bgm
int dif;-----玩家选择难度(随游戏时间自动增加)
int map;---哪张地图
int vo;-----音量
int skin;---皮肤
int AllKill;-杀了多少敌机
int GameTime;--游戏时间
int Score;---得分
}
class plane{
int range;----飞机判定圆的半径
int blood;----飞机血量
pair<int,int>coord;--飞机坐标
int vShoot;----飞机移速
bullet ForPlane;--子弹类
prop ForPlane;---道具类
}
class bullet{
pair<int,int>NowCoord;--子弹实时坐标
int Vbullet;-----------------子弹一速
int rad;------子弹判断圆的半径
int app;------外观

//子弹轨迹可以是斜着的 所以涉及极坐标和atan2
pair<int,int>InitCoord;-----射出的初始坐标
double dgree;--------------射出的角度

int ATK;---子弹攻击力
bool My;--判断是不是自己(避免自伤)

销毁函数--出屏幕/碰到就删掉
}

class Enemy::public plane{
int modol;---行为模式 迂回/冲刺
int type;------普通敌机，精英怪，boss；根据难度按比例随机生成
}

//道具
class prop{
int type;--道具种类
int CD;---道具被捡到能保存多久
int TimeDes;--道具没被捡到能存多久
}

逻辑：
判断碰到：看两个圆是否交上
生成敌机---坐标 在规定范围里面 弄个随机函数 生成；种类按难度随机生成；行为模式种类再写个函数 判断是哪种；
子弹飞行逻辑：根据射出角度和子弹速度 改变coord；
避免误伤：bullet::My==true 就不判定是否碰到；
生成道具：每杀x个 过y秒 x+y属于(a,b);


拓展：
武器：初始为机关枪，可通过拾取武器箱进行更换
道具：根据游戏时长和杀敌数生成，生成时随机赋予水平方向的初速度，向下方的初速度为固定值，触底或与敌我战机接触后消失，包括技能卡（释放导弹、展开护盾等）、强化球（永久增加攻击、移速等数值）、维修包（恢复战机生命）、武器箱（更换武器，含榴弹、激光等）
